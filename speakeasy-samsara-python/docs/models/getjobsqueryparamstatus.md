# GetJobsQueryParamStatus

A job status in STRING format. Job statuses can be one of three (ignores case): `"active", "scheduled", "completed"`  Valid values: `active`, `scheduled`, `completed`


## Values

| Name        | Value       |
| ----------- | ----------- |
| `ACTIVE`    | active      |
| `SCHEDULED` | scheduled   |
| `COMPLETED` | completed   |
