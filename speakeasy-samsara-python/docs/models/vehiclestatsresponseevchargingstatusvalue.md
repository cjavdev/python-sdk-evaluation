# VehicleStatsResponseEvChargingStatusValue

Charging status for electric and hybrid vehicles. Statuses:
 unknown - 0,
 not charging - 1,
 charging - 2.


## Values

| Name    | Value   |
| ------- | ------- |
| `ZERO`  | 0       |
| `ONE`   | 1       |
| `TWO`   | 2       |
| `THREE` | 3       |
| `FOUR`  | 4       |
