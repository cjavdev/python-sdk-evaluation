| ----------------------------- | ----------------------------- |
| `PROPERTY_OFF_DUTY_SLEEPER_`  | Property (off-duty/sleeper)   |
| `EXPLOSIVES_HAZ_MAT_ON_DUTY_` | Explosives/HazMat (on-duty)   |
