# Status

State for the Form Submission. Always returned.  Valid values: `toDo`, `submitted`, `dismissed`, `inProgress`


## Values

| Name          | Value         |
| ------------- | ------------- |
| `TO_DO`       | toDo          |
| `SUBMITTED`   | submitted     |
| `DISMISSED`   | dismissed     |
| `IN_PROGRESS` | inProgress    |
