# Language

The coaching language. The default language is English.  Valid values: `english`, `spanish`, `dutch`, `englishUk`, `italian`, `french`, `german`


## Values

| Name         | Value        |
| ------------ | ------------ |
| `ENGLISH`    | english      |
| `SPANISH`    | spanish      |
| `DUTCH`      | dutch        |
| `ENGLISH_UK` | englishUk    |
| `ITALIAN`    | italian      |
| `FRENCH`     | french       |
| `GERMAN`     | german       |
