# IftaDetailJobResponseBodyJobStatus

 The status of this job. `jobStatus` can be one of Requested, Processing, Completed, or Failed.  Valid values: `Requested`, `Processing`, `Completed`, `Failed`


## Values

| Name         | Value        |
| ------------ | ------------ |
| `REQUESTED`  | Requested    |
| `PROCESSING` | Processing   |
| `COMPLETED`  | Completed    |
| `FAILED`     | Failed       |
