# VehicleType

The type of the vehicle. Only returned for customers with commercial speed limits (CSL) enabled.  Valid values: `unset`, `passenger`, `truck`, `bus`


## Values

| Name        | Value       |
| ----------- | ----------- |
| `UNSET`     | unset       |
| `PASSENGER` | passenger   |
| `TRUCK`     | truck       |
| `BUS`       | bus         |
