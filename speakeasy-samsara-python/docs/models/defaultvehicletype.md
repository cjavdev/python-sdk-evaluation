# DefaultVehicleType

Default vehicle type (for newly added or activated vehicles).  Valid values: `off`, `automatic`, `passengerCar`, `lightTruck`, `heavyDuty`


## Values

| Name            | Value           |
| --------------- | --------------- |
| `OFF`           | off             |
| `AUTOMATIC`     | automatic       |
| `PASSENGER_CAR` | passengerCar    |
| `LIGHT_TRUCK`   | lightTruck      |
| `HEAVY_DUTY`    | heavyDuty       |
