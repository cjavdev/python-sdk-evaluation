# HealthStatus

Current overall health status of the device.  Valid values: `dataPending`, `healthy`, `needsAttention`, `needsReplacement`


## Values

| Name                | Value               |
| ------------------- | ------------------- |
| `DATA_PENDING`      | dataPending         |
| `HEALTHY`           | healthy             |
| `NEEDS_ATTENTION`   | needsAttention      |
| `NEEDS_REPLACEMENT` | needsReplacement    |
