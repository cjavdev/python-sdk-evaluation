# DvirType

Inspection type of the DVIR. Valid values: `preTrip`, `postTrip`, `mechanic`, `unspecified`.


## Values

| Name          | Value         |
| ------------- | ------------- |
| `PRE_TRIP`    | preTrip       |
| `POST_TRIP`   | postTrip      |
| `MECHANIC`    | mechanic      |
| `UNSPECIFIED` | unspecified   |
