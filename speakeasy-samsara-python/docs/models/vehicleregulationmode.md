# VehicleRegulationMode

Whether or not the vehicle is regulated, unregulated (non-CMV), or a mixed use unregulated vehicle. Valid values: `regulated`, `unregulated`, `mixed`.


## Values

| Name          | Value         |
| ------------- | ------------- |
| `REGULATED`   | regulated     |
| `UNREGULATED` | unregulated   |
| `MIXED`       | mixed         |
