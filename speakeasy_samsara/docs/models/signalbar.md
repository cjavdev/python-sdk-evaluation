# SignalBar

The cellular signal strength indicator.  Valid values: `1/4`, `2/4`, `3/4`, `4/4`, `unknown`


## Values

| Name      | Value     |
| --------- | --------- |
| `ONE_4`   | 1/4       |
| `TWO_4`   | 2/4       |
| `THREE_4` | 3/4       |
| `FOUR_4`  | 4/4       |
| `UNKNOWN` | unknown   |
