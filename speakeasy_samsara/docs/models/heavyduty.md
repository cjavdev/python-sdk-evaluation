# HeavyDuty

Harsh acceleration sensitivity settings for heavy duty vehicle.  Valid values: `unknown`, `invalid`, `off`, `low`, `normal`, `high`


## Values

| Name      | Value     |
| --------- | --------- |
| `UNKNOWN` | unknown   |
| `INVALID` | invalid   |
| `OFF`     | off       |
| `LOW`     | low       |
| `NORMAL`  | normal    |
| `HIGH`    | high      |
