# RoutesStopResponseObjectResponseBodyState

The current state of the route stop.  Valid values: `unassigned`, `scheduled`, `en route`, `skipped`, `arrived`, `departed`


## Values

| Name         | Value        |
| ------------ | ------------ |
| `UNASSIGNED` | unassigned   |
| `SCHEDULED`  | scheduled    |
| `EN_ROUTE`   | en route     |
| `SKIPPED`    | skipped      |
| `ARRIVED`    | arrived      |
| `DEPARTED`   | departed     |
