# SpeedingIntervalResponseBodySeverityLevel

Specifies the severity level of the speeding interval.  Valid values: `light`, `moderate`, `heavy`, `severe`


## Values

| Name       | Value      |
| ---------- | ---------- |
| `LIGHT`    | light      |
| `MODERATE` | moderate   |
| `HEAVY`    | heavy      |
| `SEVERE`   | severe     |
