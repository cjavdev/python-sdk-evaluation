# TachographActivityState

Tachograph activity state. Valid values: `BREAK/REST`, `WORK`, `AVAILABILITY`, `DRIVING`, `UNKNOWN`.


## Values

| Name           | Value          |
| -------------- | -------------- |
| `BREAK_REST`   | BREAK/REST     |
| `WORK`         | WORK           |
| `AVAILABILITY` | AVAILABILITY   |
| `DRIVING`      | DRIVING        |
| `UNKNOWN`      | UNKNOWN        |
