# Operation

How to evaluate the threshold.  Valid values: `GREATER`, `INSIDE_RANGE`, `LESS`, `OUTSIDE_RANGE`


## Values

| Name            | Value           |
| --------------- | --------------- |
| `GREATER`       | GREATER         |
| `INSIDE_RANGE`  | INSIDE_RANGE    |
| `LESS`          | LESS            |
| `OUTSIDE_RANGE` | OUTSIDE_RANGE   |
